# MYTHICAL — Verfügbare Spezies

Wenn kein `verfuegbare_spezies` Override in setting.yaml gesetzt ist,
stehen folgende Spezies zur Wahl:

- **Mensch** — sterblich, begrenzt, unterschätzt. In einer Welt voller Unsterblicher oft
  die gefährlichste Wahl — weil sie nichts zu verlieren haben.
- **Vampir** — unsterblich, kontrolliert, trägt Hunger wie ein zweites Bewusstsein.
  Jahrhunderte formen einen Menschen zu etwas das er selbst nicht mehr ganz versteht.
- **Werwolf** — zwei Naturen, eine Haut. Die Bestie ist kein Feind — sie ist ehrlicher
  als alles andere.
- **Nymphe / Fae** — gebunden an einen Ort oder eine Kraft. Schöner als nötig.
  Gefährlicher als sie wirken.
- **Gestaltwandler** *(generisch)* — kann eine andere Form annehmen. Was ist das wahre Gesicht?

Präsentiere die Optionen atmosphärisch — die Spannung zwischen den Naturen
ist oft interessanter als die Spezies selbst.
